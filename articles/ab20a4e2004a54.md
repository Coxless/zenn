# 動機
Azureの料金はRegion単位で異なることをしったので、主要クラウドの料金をRegionごとに比較してみた

# やること
AWS Azure GCPの主要3クラウドの料金比較を行います。各クラウドの代表的なサービスについて、同等のスペックでの料金を比較し、コストパフォーマンスを評価します。

# 比較条件
オンデマンド　仮想マシン(VM)をもとに比較する 各クラウドの汎用インスタンスを使用　vCPU 2, メモリ 8 GBで比較
- AWS: EC2 t3.large, vCPU 2, メモリ 8 GB
- Azure: Standard_B2s_v2, vCPU 2, メモリ 8 GB
- GCP: e2-standard-2, vCPU 2, メモリ 8 GB
